# ASPy - *Aurélien Stcherbinine Python*

Aurélien Stcherbinine's personal useful additions to Python.

## Installation

Clone the repository and install with pip:

~~~bash
git clone https://github.com/AStcherbinine/aspy.git
cd aspy
pip3 install .
~~~
